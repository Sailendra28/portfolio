Project-2

Path Planning And Failure Analysis Of UAV With Digital Technologies

Failure analysis and path planning of UAV is crucial for drone aviation. Path Planning has gained significant need due to its capability to simplify the process of Drone travelling. It has set up wide use in logistics and supply chain, businesses, and other associations that use the UAV. The purpose of this Path planning is to avoid the drones to crash while traveling in a dynamic and static environment. Saving UAV from crashing. The main features of the path planning is to avoid the obstacles and travel in a safe path by using Lidar sensor. The main objective is to create a safe path for Unmanned Aerial Vehicles (UAVs) to avoid highly populated areas and analyze how UAVs can be operated more safely. Any failures can cause casualties. Different UAVs that use horizontal and vertical take-off and landing are to be analyzed with different classes of wing, weight, velocity, and altitude to simulate failure due to a crash. The analysis is achieved using MATLAB Simulink and crash probability density (CPD) is estimated using Newton's laws of motion and Galileo's free fall.


Skills: MATLAB · Simulink · Problem Analysis · Critical Thinking · Team Coordination · Team Management
